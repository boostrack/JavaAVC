Required components:
 * GCC (http://www.gnu.org/software/gcc/)
 * Make (http://www.gnu.org/software/make/)
 * Libc x32 (http://www.gnu.org/software/libc/)

Install in Ubuntu:
    $ sudo apt-get install gcc make libc6-dev-i386

Install in openSUSE:
    $ sudo zypper install gcc make glibc-devel-32bit
