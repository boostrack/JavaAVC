### What it is?
JavaAVC (Java Audio/Video Codec) is FFmpeg wrapper for Java programming language. Currently supported FFmpeg 2.0.


### Requirements
 * Java 1.7 or newer.
 * *OS*: Linux or Windows, *Arch*: i386 or x86/x64.
